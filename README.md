# SWS Skeptrons Website

Official website of the **Social Welfare Skeptrons (SWS)** — Alpha Kappa Rho International Humanitarian Service Fraternity & Sorority, Region VII.

## About

- **Chapter:** Social Welfare Skeptrons (SWS)
- **Organization:** Alpha Kappa Rho
- **Founded:** August 8, 1973 (National) · 2021 (SWS Chapter)
- **Region:** Region VII, Philippines
- **Motto:** *"Truth Conquers All"*

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS / custom CSS
- Framer Motion
- Express.js (backend/API)

## Features

- Landing page with disclaimer & YouTube background audio
- Anniversary countdown to AKRho's founding (auto-updates yearly)
- Live geolocation display
- Member verification system
- News & updates
- SWS Activities gallery
- Chapter officers listing

## Development

```bash
npm install
npm run dev
```

## Deployment

Deployed on **Vercel** at [sws-skeptrons.vercel.app](https://sws-skeptrons.vercel.app)
